SpecificationsEspecificaciones

Core Architecture & Tools

Neko-Void runs on Void Linux, using runit for an extremely fast boot. Its main edition features the MATE desktop (ISOs with other environments are coming soon). The system comes packed with essential tools out-of-the-box, including Flatpak, Tinyfetch (a minimalist fetch), btop, Octoxbps, and much more.

Neko-Void corre sobre Void Linux, usando runit para un arranque extremadamente veloz. Su edición principal presenta el escritorio MATE (próximamente se crearán ISOs con otros entornos). El sistema incluye herramientas esenciales listas para usar, como Flatpak, Tinyfetch (un fetch minimalista), btop, Octoxbps y mucho más.

Performance Ready

Optimized for performance with full Vulkan support and preconfigured non-free Intel/AMD drivers, ensuring your hardware runs smoothly from the very first minute.

Optimizado para el rendimiento con soporte completo para Vulkan y drivers non-free de Intel/AMD preconfigurados, asegurando que tu hardware funcione sin problemas desde el primer minuto.

Kasha Installer & Post-Install

Our custom installer simplifies the Void Linux installation process (user, network, partitions). Once installed, a built-in post-install application installer allows you to automatically configure the system with one click for video editing, music production, design, gaming, office work, and more.

Nuestro instalador personalizado simplifica el proceso de instalación de Void Linux (usuario, red, particiones). Una vez instalado, un instalador de aplicaciones post-instalación integrado te permite configurar el sistema automáticamente con un clic para edición de video, producción musical, diseño, gaming, ofimática y más.

Neko-Void

Neko-Void is an unofficial Void Linux respin. It currently features a preconfigured MATE desktop environment (our main edition, though other ISOs with more desktops will be created) with selected software for a complete experience. It includes a powerful post-install assistant to tailor the system to your needs (gaming, design, music production, etc.), plus pre-installed Flatpak, Octoxbps, Tinyfetch, and btop. All backed with full Vulkan and Pipewire support.

Neko-Void es un respin no oficial de Void Linux. Actualmente, presenta un entorno de escritorio MATE preconfigurado (nuestra edición principal, aunque se crearán otras ISOs con más escritorios) con software seleccionado para ofrecer una experiencia completa. Incluye un potente asistente post-instalación para adaptar el sistema a tus necesidades (gaming, diseño, producción musical, etc.), además de contar con Flatpak, Octoxbps, Tinyfetch y btop preinstalados. Todo respaldado con soporte completo para Vulkan y Pipewire.

Why it was createdPor qué se creó

The project originally started as a personal hobby. Over time, it evolved into a structured project aimed at offering an easy-to-install and user-friendly Void Linux version for the end user.

El proyecto se originó inicialmente como un hobby personal. Con el tiempo, evolucionó hasta convertirse en un proyecto estructurado con el objetivo de ofrecer una versión de Void Linux que sea fácil de instalar y utilizar para el usuario final.

Target AudiencePúblico dirigido

This distribution is aimed at users who want a lightweight and robust operating system, easily customizable for their specific workflow without the usual complications of a manual setup from scratch.

Esta distribución está dirigida a usuarios que desean un sistema operativo ligero y robusto, fácilmente adaptable a su flujo de trabajo específico sin las complicaciones habituales de una configuración manual desde cero.
